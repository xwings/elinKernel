# elinKernel 快速开始

> **🚧 翻译进行中** - 本文档正在翻译中，详细内容请参考 [英文版本](../en/getting-started.md)。

本指南介绍如何从源代码设置、编译和运行 elinKernel。

## 先决条件

- **Rust 工具链** 包含 `riscv64gc-unknown-none-elf` 目标
- **QEMU** 支持 RISC-V (`qemu-system-riscv64`)
- **mkfs.fat** 用于磁盘镜像格式化（可选）

## 快速开始

```bash
# 编译并运行 elinKernel
./build.sh
./run.sh
```
